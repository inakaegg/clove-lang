# 用語集

- 更新日: 2026-01-14

- **Form**: Reader が読み取った構文木ノード（位置情報つき）
- **Value**: 評価器が扱う実行時値
- **Reader**: 文字列/ファイルを Form に変換する層（ここで構文糖を展開）
- **special form**: 評価器が “普通の関数呼び出し” として扱わない構文
- **dot-chain**: `x.(f ?)` のような “? プレースホルダ” を含む段階式の連結
- **indexer**: `x[0]`, `x[0 1]`, `x[0||d]` のようなアクセス構文糖
- **foreign blocks**: `$rb{...}`, `$py{...}`, `${...}`（Ruby の省略記法）のように外部言語を埋め込む記法
- **promise/future/task/agent**: Clove の非同期/並行ハンドル
