# SDL2 Intro Example

This directory contains minimal entry examples for trying Clove's SDL2 plugin.
For more examples, see the examples in the clove-sdl2 repository.

## Examples

- `01_events_input.clv`: basics of event polling and key input

## Run

```bash
clove --allow-native-plugins --main examples/sdl2/01_events_input.clv
```

## Dependencies

- clove-sdl2
