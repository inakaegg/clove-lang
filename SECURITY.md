# Security Policy

## Supported Versions

At this time, we support **only the latest main branch / latest release**.

## Reporting

Please do not file public issues for security problems. Report privately if possible.

- Preferred: GitHub Security Advisory (Private)
- If you need a contact point: https://github.com/inakaegg

## Information to Include

- Impact (what can happen)
- Reproduction steps (minimal)
- Affected scope (OS / version / configuration)
- If possible, workaround or patch ideas

## Response Timeline

We aim to **respond within 7 days**.
If a fix is needed, we will coordinate disclosure timing and communication.
